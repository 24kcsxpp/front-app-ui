## Attributes

| 参数  | 说明           | 类型 | 可选值 | 默认值 |
| ----- | -------------- | ---- | ------ | ------ |
| class | 自定义面板样式 | any  | —      | -      |

## Panel Attributes

| 参数  | 说明                                            | 类型   | 可选值 | 默认值 |
| ----- | ----------------------------------------------- | ------ | ------ | ------ |
| class | 自定义面板样式                                  | any    | —      | -      |
| title | 索引面板的标题，若不填写则使用 `index` 参数的值 | string | —      | -      |
| index | 索引标志（用于标识当前面板的索引类别）          | string | —      | —      |
